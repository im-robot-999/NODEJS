HTTP Server File Commenter

program_1.js
Node.js HTTP server that returns content of requested file with each line commented out. Starts server on 127.0.0.1:3000 and processes file requests.

Key Features:

• HTTP server on port 3000 • File reading with error handling • Automatic line commenting with "// " • URL parsing for file requests • Plain text response format

Example Output:
